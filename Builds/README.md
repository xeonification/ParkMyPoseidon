# ParkMyPoseidon

Builds must be placed in this directory maintaining this format :

[ReleaseVersions]
PA - Pre Alpha
A - Alpha
B - Beta
PAUI - Pre Alpha UI 
AUI - Alpha UI
BUI - Beta UI
RC1 - Release Candidate 1
RC2 - Release Candidate 2
RCF - Release Candidate Final

PMP(DateOfTheBuild)(ReleaseVersion)
Example : PMP21092015PA
